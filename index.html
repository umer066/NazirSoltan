Mobile Wallet with Merchant payment

Technical Description

A mobile wallet merchant payment system is programmed to make it simple for a user to make cashless purchases from merchants. Only cashless methods of making payments and conducting transactions are available to people in this generation. With the use of this application, customers can effortlessly make cashless payments to retailers. Customers must add money to their mobile wallets to purchase from merchants. For this project, we will use HTML, CSS, Bootstrap, and JavaScript for the frontend, and for the backend, we will use Python, Django, Payture, and Flask.
